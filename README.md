# Jugal Rajput — Portfolio

Live: [jugalrajputportfolio.netlify.app](https://jugalrajputportfolio.netlify.app/)

## Stack

- React 18
- Framer Motion
- Swiper (3D Carousel)
- Lucide Icons
- EmailJS
- Vite

## Run locally

```bash
nvm use 18
npm install
npm run dev
```

## Build

```bash
npm run build
```
